本地

一、下载Node.js，点击LTS版本

二、安装Git

一直点【下一步】就安装完成了，完成后开始菜单出现【Git Bash】

之后的命令输入都在【Git Bash】这个窗口

Ctrl + C 断开命令

三、安装cnpm提高速度

输入cd blog新建一个文件夹，或者直接在盘里新建一个文件夹后【Git Bash Here】

输入node -v查询nodejs版本，npm -v查询vpm版本，是否安装成功。

在【Git Bash】窗口输入npm install -g cnpm --registry=https://registry.npm.taobao.org

在【Git Bash】窗口输入cnpm验证是否成功

四、用cnpm安装Hexo

在【Git Bash】窗口输入cnpm install -g hexo-cli

安装完成后用hexo -v来验证是否安装成功

看看当前路径命令pwd

输入md blog后，可在对应路径找到一个名为【blog】的文件夹。输入cd blog进入【blog】，再输入chdir看一下路径是对的没问题。

如果搭建博客过程中有问题，大不了把文件夹干掉重来

输入hexo init，建立整个项目

五、启动

输入hexo server，把localhost:4000复制到浏览器出现下图，说明本地环境搭建完成。

hexo new "article name"新建一篇文章（存入blog/source/_posts）

cd source/_posts 定位到该目录

ls / dir查找该目录下文件，出现刚才新建的文章

vim name.md进入vim编辑页面，先按【Esc】再输入:wq退出

pwd 查看当前位置发现进了两层

cd ../..退回两层，再pwd 查看当前位置为【blog】

ls查找该目录下文件，hexo clean清理一下，hexo generate生成一下，再hexo server启动一下

在线

  1. (Github](https://github.com/)登陆后，新建仓库（Create a new repository）名为：用户名.github.io。

随便写个描述，【Create repository】，复制【HTTPS】地址

  1. 回到 【Git Bash】输入cnpm install --save hexo-deployer-git部署插件

  2. 设置一下 【blog】里的【_config.yml】（站点配置文件）：用vim _config.yml打开，翻到最后面的【Deployment】，type: :’git’，回车写上仓库地址repo: 【HTTPS】地址，回车写上branch: master

  3. 保存退出

    冒号后一定要有空格！

  4. hexo deploy部署到服务器，正常会跳出登录页面，

  5. 如没有登陆页面则输入空两格git config --global user.email xxx回车再输入空两格git config --global user.name xxx ，再输入hexo deploy，就会弹出登陆页面了，去设置的developing setting新建一个token。

  6. 刷新一下仓库，多出一些文件即代表成功啦！就可以访问自己的网站了https://xxxx.github.io

主题

推荐：litten/hexo-theme-yilia: 一个简洁优雅的hexo主题 A simple and elegant theme for hexo. (github.com)

  1. pwd在【blog】路径，输入git clone https://github.com/iTimeTraveler/hexo-theme-hiero.git themes/hiero

    或者直接把主题下载到blog/themes文件夹里

手打一遍加载速度比较快

  1. pwd 然后继续cd themes/打开【themes】文件夹，ls -l找到刚才下载的主题。

  2. 回到【blog】目录下ls -l展开所有项，修改vim _config.yml，找到【theme】（在最后面的Deployment上一项）修改为新下载的【主题名】

  3. hexo clean 清理一下，hexo generate重新生成，hexo server本地运行一下，hexo deploy在线应用。

  4. 在hexo deploy这步出现了错误error：spawn failed...，根据此文hexo发生error：spawn failed错误的解决方法_HTL2018的博客-CSDN博客成功解决。

    • 删除【blog】下的【.deploy_git】文件夹

    • 输入git config --global core.autocrlf false

    • 然后，依次执行：
      hexo clean
      hexo generate
      hexo deploy

成功更换主题🤩
